{% raw %}List of the Principal Streets, Squares,{% endraw %}
{% raw %}Public Buildings, Bridges, etc., Of Berlin.{% endraw %}
{% raw %}The large Map of Berlin, on the scale of 1 :20,000, is divided{% endraw %}
{% raw %}into three sections, of which the uppermost is coloured brown, tin:{% endraw %}
{% raw %}central red, and the lowest gray. Each section contains 42 num¬{% endraw %}
{% raw %}bered squares. In the following index the capital letters B, R, Gr,{% endraw %}
{% raw %}following the name of a street or building, refer to the different{% endraw %}
{% raw %}sections, while the numbers correspond with those of the squares in{% endraw %}
{% raw %}each section. Thus, Admiral-Strasse will be found on the grey{% endraw %}
{% raw %}section, square 29. If the name sought for is also indicated on the{% endraw %}
{% raw %}Plan of the Inner Town printed in red (in the scale of 1:12,500;{% endraw %}
{% raw %}p. 54), this is shown by an I before the first perpendicular line.{% endraw %}
{% raw %}The numbering of the squares is so arranged, that squares in dif¬{% endraw %}
{% raw %}ferent sections bearing the same number adjoin each other. Thus,{% endraw %}
{% raw %}square 15 on the brown section finds its continuation towards the{% endraw %}
{% raw %}X. in square 15 on the red section.{% endraw %}
{% raw %}The squares will also be useful for calculating distances, each{% endraw %}
{% raw %}side of a square being exactly a kilometre (1093 yds., or about{% endraw %}
{% raw %}2/g M.), while the diagonals if drawn would be 1531 yds.{% endraw %}
{% raw %}The large Italics (N, O, S, W, C, etc.) following the names of the{% endraw %}
{% raw %}streets indicate the postal district to which the street belongs.{% endraw %}
{% raw %}Abbreviations or names of districts and villages in the environs of{% endraw %}
{% raw %}Berlin: Box. = Boxhagen; Char. = Charlottenburg; Grun. = Grunewald;{% endraw %}
{% raw %}Hal. = Halensee; Plo. = Plbtzensee; Rix. = Rixdorf; Rum. = Rummels-{% endraw %}
{% raw %}burg; Schmar. = Schmargendorf; Schon. = Schoneberg; Trep. = Treptow;{% endraw %}
{% raw %}Weis. = Weissensee; Wilm. = Wilmersdorf.{% endraw %}
