{% raw %}a. Railway Stations. Arrival and Departure.{% endraw %}
{% raw %}Railway Stations. There are Ave Terminus Railway Sta¬{% endraw %}
{% raw %}tions at Berlin, exclusive of the 'Stadtbahn' (see below). 1. Anhalt{% endraw %}
{% raw %}Station (PL G, 19), for Dresden, Leipzig, Halle (Austria, Bavaria,{% endraw %}
{% raw %}Thuringia, Frankfort on the Main). — 2. Potsdam Station (PI. G, 19),{% endraw %}
{% raw %}for Potsdam, Magdeburg, Cassel, Cologne, and the Lower Rhine{% endraw %}
{% raw %}(junction with the 'Stadtbahn' at Potsdam). — 3. Lehrte Station{% endraw %}
{% raw %}(PL R, 18, 21), for all trains to Hamburg and slow trains to Lehrte,{% endraw %}
{% raw %}Hanover, Bremen, etc. (junction with the 'Stadtbahn' at Spandau).{% endraw %}
{% raw %}— 4. Stettin Station (PI. R, 24), for Rostock (and Copenhagen),{% endraw %}
{% raw %}Stettin, and Dantsic (via Stargard). — 5. Gorlitz Station (PL G, 32),{% endraw %}
{% raw %}for the Spreewald, Gorlitz, and the Giant Mts. (junction with the{% endraw %}
{% raw %}'Stadtbahn' at Nieder-Schoneweide). — The following five Stations{% endraw %}
{% raw %}of the 'Stadtbahn' (see p. 12) are also used for general traffic:{% endraw %}
{% raw %}1. Charlottenburg Station (PI. G, 4). — 2. Zoological Garden{% endraw %}
{% raw %}Station (PL G, 10). —3. Friedrich-Strasse Station (PI. R, 23;{% endraw %}
{% raw %}with the heaviest traffic). — 4. Alexander-Platz Station (PI. R, 26).{% endraw %}
{% raw %}— 5. Silesian Station (PL R, 31). — The express trains from Han¬{% endraw %}
{% raw %}over, Bremen, Cologne and all the trains from Nordhausen-Frank-{% endraw %}
{% raw %}fort on the Main, Kdnigsberg, Posen, and Breslau arrive at these{% endraw %}
{% raw %}stations.{% endraw %}
{% raw %}The Custom House, Examination of registered luggage, arriving from{% endraw %}
{% raw %}abroad at the stations of the Stadtbahn or at the Stettin Station, is at the{% endraw %}
{% raw %}Friedrich-Strasse Station (daily 11-12 a.m.). — Head Custom House, Alt-{% endraw %}
{% raw %}Moabit 145.{% endraw %}
{% raw %}Arrival. — Cabs are to be found on the S. side at the stations{% endraw %}
{% raw %}of the Stadtbahn (except at Charlottenburg and Alexander-Platz),{% endraw %}
{% raw %}and on the arrival (right) side at the terminus stations (except at{% endraw %}
{% raw %}the Gorlitz Station where they are straight in front). A policeman,{% endraw %}
{% raw %}posted at the exit from each railway-station, hands the traveller a{% endraw %}
{% raw %}metal ticket with the number of a cab, on his stating whether he{% endraw %}
{% raw %}wishes a cab ('Droschke') or a luggage cab (iGepack-Droschke,J.{% endraw %}
{% raw %}Travellers with luggage should entrust the summoning of the vehicle{% endraw %}
{% raw %}to the porter (20 pf. for luggage up to 55 lbs., 10 pf. for every 55 lbs.{% endraw %}
{% raw %}more). Cab fares see p. 23; in addition to the fare a charge of{% endraw %}
{% raw %}25 pf. is made for the ticket securing the cab. 'Gepack-Droschken'{% endraw %}
{% raw %}House numbers in Berlin generally begin at the end of the street{% endraw %}
{% raw %}nearest where the old town-gates originally stood, and thence run conse¬{% endraw %}
{% raw %}cutively along the right-hand side of the street, returning on the left-{% endraw %}
{% raw %}hand side, so that the highest number is opposite to No. 1.{% endraw %}
